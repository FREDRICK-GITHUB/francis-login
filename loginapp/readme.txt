1.Create a database called loginapp
2.Import the database included in this folder (loginapp.sql) to the db you have just created.
3.Place the (loginapp) folder in the respective www folder in your machine
4.Import the android project (LoginApp) in your android studio and remember to set the correct ip address in the (BackgroundTask.java) file
5.The login_test.html and register_test.html are used for testing whether the connection to the db is successful
Good Luck!
